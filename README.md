# 🏆 منصة المسابقات — دليل الاستخدام

## 📁 هيكل الملفات
```
quiz-platform/
├── index.html           ← الصفحة الرئيسية
├── competition.html     ← صفحة النماذج
├── quiz.html            ← صفحة الاختبار
├── shared.css           ← التصميم المشترك
├── shared.js            ← الدوال المشتركة
├── [صورة الشعار]       ← ضع صورتك هنا
└── data/
    ├── meta.json        ← بيانات المسابقات (بدون أسئلة)
    ├── q-1-1.json       ← مسابقة 1، نموذج 1
    ├── q-1-2.json       ← مسابقة 1، نموذج 2
    ├── q-1-3.json       ← مسابقة 1، نموذج 3
    ├── q-2-1.json       ← مسابقة 2، نموذج 1
    ├── q-2-2.json       ← مسابقة 2، نموذج 2
    ├── q-2-3.json       ← مسابقة 2، نموذج 3
    ├── q-3-1.json       ← مسابقة 3، نموذج 1
    ├── q-3-2.json       ← مسابقة 3، نموذج 2
    └── q-3-3.json       ← مسابقة 3، نموذج 3
```

## ▶️ كيف تشغّله

**لا تفتح index.html مباشرة!** يحتاج سيرفر محلي:

### VS Code (الأسهل):
1. ثبّت إضافة "Live Server"
2. كليك يمين على index.html → Open with Live Server

### Python:
```bash
cd quiz-platform
python -m http.server 8000
```
ثم افتح: http://localhost:8000

## ➕ إضافة نموذج جديد

1. أنشئ ملف `data/q-[رقم المسابقة]-[رقم النموذج].json`
2. أضف سطراً في `meta.json` تحت المسابقة المناسبة

## 🎨 تغيير الشعار

في الثلاث صفحات، غيّر اسم الصورة في:
```html
<img src="اسم_صورتك.jpeg" alt="شعار">
```
